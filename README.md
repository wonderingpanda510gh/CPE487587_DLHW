# Deeplearning HW

This repository contains scripts and experimental code for CPE 587 Deep Learning Home Work.
The code is intended for homework.

## Environment Setup

We recommend using a virtual environment.
```
git clone https://github.com/wonderingpanda510gh/CPE587_HW.git
cd CPE587_HW
uv venv
source .venv/bin/activate
uv pip install -r requirements.txt
uv pip install -e .
```

Python version tested:
```
Python 3.12.3
```

## How to Run
Run the main script from the project root:

For the HW01:
```
python scripts/binaryclassification_impl.py
```
The script will automatically:

- create a output directories

- save trained weights and save generated PDF
